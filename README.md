## 1. Visão geral do projeto

Este repositório contém a implementação em **HTML e CSS** do protótipo criado no Figma para a loja fictícia **Women’s Shopping**.  
O objetivo foi aplicar os conceitos estudados em aula e publicar o site usando GitHub Pages.

---

## 2. Páginas incluídas

- `index.html` — Página inicial com cards de produtos (mínimo de 5 itens).  
- `html/detalhes.html` — Página de detalhes de produto (imagem principal, título, descrição, preço e seção com outros produtos).  
- `html/contato.html` — Página de contato com formulário (nome, e-mail, assunto, mensagem e botão de envio).  
- `css/style.css` — Estilos do site.  
- `img/` — Pasta com imagens utilizadas no site.
